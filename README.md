Portfolio Website
A standalone, single-file HTML portfolio showcasing AI prototypes, articles, posts, and public appearances.


##Overview
This is a self-contained portfolio website built as a single HTML file with no build process required. It features a modern, responsive design with dark mode support and showcases professional work in AI, business analysis, and enterprise architecture.

##Tech Stack

HTML5 - Single-file architecture
React 18.2.0 - UI framework (loaded via ES modules from esm.sh)
Tailwind CSS - Utility-first styling (loaded via CDN)
Lucide React 0.460.0 - Icon library (loaded via ES modules)

##Features

###Core Functionality

**Category Filtering** - Filter by Prototypes, Articles, Posts, or Public Appearances
**Platform Filtering** - Toggle between Claude Artifacts and NotebookLM resources
**Subcategory Navigation** - Browse TOGAF EA, TOGAF BA, CBAP Prep, and Technical resources
**Global Search** - Real-time search across all content within selected categories
**Alphabetical Sorting** - Automatic sorting of filtered results by title
**Dark Mode** - System preference detection with manual toggle (Sun/Moon icon)
**Responsive Design** - Mobile-first with hamburger menu and adaptive layouts

###User Experience

**Sticky Navigation** - Header remains fixed during scroll
**Collapsible Hero** - Minimize stats/bio section for focused browsing
**Clickable Stats** - Direct navigation to categories via interactive metric cards
**Scroll-to-Top Button** - Appears after scrolling 300px
**Smooth Scrolling** - CSS-based smooth navigation throughout
**Empty State UI** - User-friendly messaging when no results found

###Visual Design

**Color-coded Categories** - Unique color schemes per content type (Indigo/Emerald/Blue/Purple)
**Badge System** - Visual indicators for Video/Article/Platform types
**Interactive Cards** - Hover effects with shadows, transforms, and color transitions
**Custom Animations** - Fade-in-down, bounce, pulse, rotate, and scale effects
**Hidden Scrollbars** - Clean overflow handling on subcategory navigation
**Professional Certification Badges** - CBAP®, PMP®, TOGAF® EA Practitioner pills

###External Integration

**Download CV** - Direct link to hosted resume PDF
**LinkedIn Connect** - Prominent social profile integration
**External Links** - All resources open in new tabs with noopener noreferrer security

## Content Organization

**30+ AI Prototypes** - TOGAF/CBAP study tools across Claude & NotebookLM platforms
**8 Articles** - AI strategy, organizational transformation, and technical deep dives
**8 LinkedIn Posts** - Industry insights on AI ethics, hallucinations, and trends
**3 Public Appearances** - Video interviews and media features

## Usage
Simply open index.html in any modern browser. No build process, server, or dependencies required.

# Clone the repository
git clone https://github.com/yourusername/portfolio.git

# Navigate to the directory
cd portfolio

# Open in browser
open index.html
